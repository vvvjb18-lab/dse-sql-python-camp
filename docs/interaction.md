# DSE SQL 訓練營 - 交互設計規劃

## 網站結構

### 主要頁面
1. **首頁 (index.html)** - 主要訓練平台
   - Hero區域：簡潔的標題和開始按鈕
   - SQL編輯器區域：左側題目，右側編輯器
   - 進度追蹤：實時顯示學習進度
   - 成就系統：解鎖的徽章和證書

2. **學習指南 (guide.html)** - 理論知識
   - DSE SQL課程大綱
   - 語法參考手冊
   - 常見錯誤指南
   - 考試技巧分享

3. **練習題庫 (practice.html)** - 額外練習
   - 分類題目瀏覽
   - 難度篩選器
   - 收藏夾功能
   - 錯題重做

4. **成績追蹤 (progress.html)** - 學習分析
   - 詳細統計數據
   - 強弱項分析
   - 學習建議
   - 考試準備評估

## 核心交互功能

### SQL訓練系統
- **即時編輯器**：支援語法高亮和自動完成
- **智能檢查**：即時語法錯誤提示
- **分步提示**：按需求提供解題提示
- **多表查詢**：支援JOIN操作的視覺化練習
- **結果對比**：預期結果與實際結果的並排顯示

### 錯誤指導系統
- **錯誤分類**：語法錯誤、邏輯錯誤、性能問題
- **詳細解釋**：每種錯誤的具體說明和修正建議
- **相關練習**：針對性推薦類似題目進行鞏固
- **學習路徑**：根據錯誤調整後續學習內容

### 進度追蹤系統
- **知識點地圖**：視覺化顯示掌握程度
- **學習曲線**：每日/每週學習時間統計
- **題目完成率**：不同難度題目的完成情況
- **模擬考試**：定期模擬測試和成績記錄

### 成就系統
- **等級系統**：根據完成題目數量和準確率升級
- **技能徽章**：掌握特定SQL技能的證明
- **連續學習**：連續登入和學習的獎勵
- **挑戰模式**：限時挑戰和高分榜

## 階梯式學習設計

### 初級階段（基礎查詢）
1. **SELECT基礎**：簡單查詢和列選擇
2. **WHERE條件**：基本比較運算符和邏輯運算
3. **排序操作**：ORDER BY的使用
4. **基本函數**：COUNT、SUM、AVG等聚合函數

### 中級階段（數據操作）
1. **多表連接**：INNER JOIN、LEFT JOIN的使用
2. **子查詢**：簡單子查詢和相關子查詢
3. **數據分組**：GROUP BY和HAVING子句
4. **數據修改**：INSERT、UPDATE、DELETE操作

### 高級階段（複雜查詢）
1. **複雜連接**：多表連接和自連接
2. **嵌套查詢**：多層子查詢和WITH子句
3. **數據分析**：窗口函數和高級聚合
4. **性能優化**：索引使用和查詢優化

## 用戶體驗設計

- **桌面端**：左右分欄佈局，充分利用屏幕空間
