## 📱 詳細設計規格

### 1. 頂部導航欄

#### 核心功能區
- **服務狀態總覽**（左側）
  - 狀態指示器：🟢 運行中 / 🔴 已停止 / 🟡 重啟中
  - 進程 ID：顯示 PID，點擊複製
  - 服務端口：可切換本地/公網模式
  - 健康檢查：實時狀態，點擊詳情

#### 快速操作按鈕組（中間）
- **智能服務控制按鈕**：
  - 根據狀態動態顯示：▶ 啟動 / ■ 停止 / ↻ 重啟
  - 單一按鈕，智能切換
- **緊急操作**：快速停止/重啟（紅色強調）
- **健康檢查**：一鍵診斷，帶結果預覽

#### 全域功能（右側）
- **全局搜索**：跨功能搜索（日誌、用戶、文件）
- **通知中心**：系統通知、錯誤警報、操作反饋
- **用戶設定**：外觀主題、語言、自動刷新頻率

## 🎨 視覺設計系統

### 色彩系統

#### 主色調
- **深藍** (#2c3e50)：主要背景、側邊欄
- **青藍** (#3498db)：主要按鈕、鏈接

#### 狀態色
- **正常**：青綠 (#2ecc71) 🟢
- **警告**：琥珀 (#f39c12) 🟡
- **嚴重**：紅 (#e74c3c) 🔴
- **資訊**：藍 (#3498db) 🔵

#### 背景層級
- **主要內容**：#ffffff（白色）
- **側邊欄**：#f8f9fa（淺灰）
- **狀態面板**：#2c3e50（深藍，深色模式）

### 排版系統

#### 字體層級
- **標題**：18px, font-weight: 600
- **副標題**：14px, font-weight: 500
- **內文**：13px, line-height: 1.5
- **代碼/日誌**：12px, monospace

#### 間距系統
- **基準單位**：8px
- **間距**：8/16/24/32/48px

### 互動反饋

#### 按鈕狀態
- **正常**：標準樣式
- **懸停**：顏色加深，陰影
- **點擊**：顏色變淺，陰影消失
- **禁用**：灰色，不可點擊

#### 載入指示
- 所有異步操作顯示進度指示
- 使用旋轉圖標或進度條

#### 通知系統
- **成功**：綠色 toast，自動消失
- **錯誤**：紅色 toast，需手動關閉
- **警告**：黃色 toast，自動消失
- **資訊**：藍色 toast，自動消失

## 📚 技術實現

### Tkinter 限制與解決方案

#### 限制
- Tkinter 不支援現代 CSS 樣式
- 響應式設計需要手動實現
- 複雜佈局需要更多代碼

#### 解決方案
- **自定義樣式**：使用 ttk.Style() 自定義主題
- **佈局管理**：使用 grid 和 pack 混合佈局
- **組件化**：將功能模組化，便於維護
- **顏色常量**：定義顏色常量，統一管理

### 實現策略
1. **保留原有功能**：確保所有功能正常運作
2. **逐步遷移**：分階段實現新設計
3. **向後兼容**：保留原文件作為備份
4. **測試驗證**：每個階段進行測試
