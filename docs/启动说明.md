# DSE SQL 訓練營 - 啟動說明

## 快速啟動

### 方式一：僅前端（無AI功能）

1. 直接打開 `index.html` 文件
   - 雙擊 `index.html` 文件
   - 或在瀏覽器中打開：`file:///路徑/index.html`

2. 推薦使用本地服務器（避免跨域問題）
   ```bash
   # 使用Python內置服務器
   python -m http.server 8000
   
   # 或使用Node.js的http-server
   npx http-server -p 8000
   ```
   
   然後在瀏覽器中訪問：`http://localhost:8000`

### 方式二：完整功能（包含AI助手）

#### 步驟1：安裝Python依賴

```bash
pip install -r requirements.txt
```

#### 步驟2：配置API密鑰

編輯 `config.py` 文件，將 `YOUR_API_KEY` 替換為您的智譜AI API密鑰：

```python
ZHIPU_API_KEY = "your_actual_api_key_here"
```

#### 步驟3：啟動後端服務

```bash
python app.py
```

服務將在 `http://localhost:5000` 啟動

#### 步驟4：啟動前端服務

打開新的終端窗口，運行：

```bash
# 使用Python內置服務器
python -m http.server 8000
```

#### 步驟5：訪問網頁

在瀏覽器中打開：`http://localhost:8000/index.html`

## 故障排除

### 問題1：AI功能無法使用
- 確保後端服務正在運行：`python app.py`
- 檢查API密鑰是否正確配置
- 查看瀏覽器控制台是否有錯誤

### 問題2：跨域錯誤
- 使用本地服務器而不是直接打開HTML文件
- 確保後端服務的CORS已啟用

### 問題3：頁面無法加載
- 檢查文件路徑是否正確
- 確保所有依賴文件都在同一目錄
- 使用瀏覽器開發者工具查看錯誤信息

## 開發模式

### 同時啟動前端和後端

創建 `start.bat` (Windows) 或 `start.sh` (Linux/Mac)：

**Windows (start.bat):**
```batch
@echo off
start cmd /k "python app.py"
timeout /t 2
start cmd /k "python -m http.server 8000"
echo 後端服務: http://localhost:5000
echo 前端服務: http://localhost:8000
echo 請在瀏覽器中打開: http://localhost:8000/index.html
pause
```

**Linux/Mac (start.sh):**
```bash
#!/bin/bash
python app.py &
sleep 2
python -m http.server 8000 &
echo "後端服務: http://localhost:5000"
echo "前端服務: http://localhost:8000"
echo "請在瀏覽器中打開: http://localhost:8000/index.html"
```

## 注意事項

1. **API密鑰安全**：請勿將API密鑰提交到公共代碼倉庫
2. **端口占用**：如果5000或8000端口被占用，請修改端口號
3. **瀏覽器兼容**：推薦使用Chrome、Firefox或Edge瀏覽器
4. **網絡連接**：AI功能需要網絡連接訪問智譜AI API

## 技術支持

如有問題，請檢查：
- Python版本（建議3.7+）
- 依賴包是否正確安裝
- API密鑰是否有效
- 網絡連接是否正常
