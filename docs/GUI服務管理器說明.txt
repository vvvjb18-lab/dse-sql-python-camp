========================================
DSE SQL 訓練營 - GUI 服務管理器
========================================

已成功創建 Linux 本地圖形化服務管理程序！

📁 創建的文件：
----------------
1. service-manager.py          - 主程序文件（GUI 應用）
2. start-service-manager.sh    - 啟動腳本
3. install-gui-dependencies.sh - 依賴安裝腳本
4. dse-sql-service-manager.desktop - 桌面圖標文件
5. 服務管理器使用說明.md       - 詳細使用說明
6. GUI服務管理器快速指南.md    - 快速啟動指南

🚀 快速開始：
----------------
1. 安裝依賴：
   ./install-gui-dependencies.sh

2. 啟動服務管理器：
   python3 service-manager.py

✨ 主要功能：
----------------
✓ 服務啟動/停止/重啟控制
✓ 實時服務狀態監控
✓ 健康檢查功能
✓ 實時日誌查看器
✓ 配置編輯器（API 密鑰管理）
✓ 快速訪問按鈕（一鍵打開網頁）
✓ 系統信息顯示

🎯 使用場景：
----------------
- 啟動和停止後端服務
- 監控服務運行狀態
- 查看服務日誌
- 編輯配置文件
- 快速訪問各個網頁頁面

📝 注意事項：
----------------
1. 首次使用需要安裝 Tkinter：
   sudo apt-get install python3-tk

2. 確保 Python 3.6+ 已安裝

3. 服務管理器窗口關閉不會自動停止服務，
   請先點擊「停止服務」按鈕

4. 修改配置後需要重啟服務才能生效
