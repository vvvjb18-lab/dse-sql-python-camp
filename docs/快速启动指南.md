## ⚠️ 如果仍然無法連接

1. **檢查後端是否運行**：
   ```bash
   netstat -ano | findstr :5000
   ```

2. **查看後端終端輸出**：
   - 如果有錯誤信息，請查看具體錯誤
   - 常見錯誤：
     - API密鑰未配置（不影響服務啟動，只是AI功能不可用）
     - 端口被占用

3. **重啟服務**：
   - 停止當前服務（Ctrl+C）
   - 重新運行：`python app.py`
