# DSE SQL 訓練營網站設計規範

## 設計理念

### 色彩系統
- **主色調**: 深海藍 (#1e3a8a) - 象徵知識的深度和專業性
- **輔助色**: 薄荷綠 (#10b981) - 代表成長和成功
- **強調色**: 琥珀橙 (#f59e0b) - 用於重要提示和成就標記
- **中性色**: 石墨灰 (#374151) - 用於文本和邊框
- **背景色**: 純白 (#ffffff) 和淺灰 (#f8fafc)

### 字體系統
- **標題字體**: Noto Serif SC (粗體) - 權威感和可讀性
- **正文字體**: Noto Sans SC (常規) - 現代感和清晰度
- **代碼字體**: JetBrains Mono - 等寬字體，適合SQL代碼顯示

### 視覺語言
- **極簡主義**: 乾淨的界面，專注於學習內容
- **卡片設計**: 使用陰影和圓角創造層次感
- **漸變效果**: 微妙的漸變增加視覺吸引力
- **圖標系統**: 使用線性圖標，保持一致的視覺風格

## 視覺效果

### 使用的庫和效果
- **Anime.js**: 用於按鈕動畫和頁面轉場
- **ECharts.js**: 用於學習進度可視化
- **p5.js**: 用於背景粒子效果
- **Splide.js**: 用於題目輪播
- **Matter.js**: 用於成就系統的物理動畫

### 動畫設計
- **微交互**: 按鈕懸停時的輕微縮放和陰影變化
- **進度指示**: 使用動畫條形圖顯示學習進度
- **成功反饋**: 完成題目時的粒子爆炸效果
- **錯誤提示**: 溫和的搖晃動畫

### 背景效果
- **粒子系統**: 使用p5.js創建浮動的代碼片段背景
- **漸變流動**: 使用CSS動畫創建微妙的色彩流動
- **幾何圖案**: 使用SVG創建抽象的幾何裝飾

### 標題效果
- **打字機效果**: 主標題使用逐字顯示動畫
- **顏色循環**: 關鍵詞的顏色漸變循環
- **發光效果**: 重要標題使用微妙的光暈效果

### 滾動動畫
- **淡入效果**: 內容區塊在滾動時淡入顯示
- **視差滾動**: 背景元素使用輕微的視差效果
- **進度指示**: 側邊進度條跟隨滾動進度
