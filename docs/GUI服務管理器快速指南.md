# GUI 服務管理器 - 快速啟動指南

## 🚀 快速開始

### 第一步：安裝依賴

```bash
cd "/home/yivh/web surver"
./install-gui-dependencies.sh
```

或手動安裝（Ubuntu/Debian）：
```bash
sudo apt-get install python3-tk
```

### 第二步：啟動服務管理器

```bash
python3 service-manager.py
```

或使用啟動腳本：
```bash
./start-service-manager.sh
```

## 📱 界面功能

### 左側控制面板

1. **服務狀態**
   - 顯示當前服務運行狀態
   - 顯示進程 ID 和端口號

2. **控制按鈕**
   - ▶ 啟動服務
   - ■ 停止服務
   - ↻ 重啟服務
   - 🔍 健康檢查

3. **快速訪問**
   - 一鍵打開各個網頁頁面

4. **配置管理**
   - ⚙️ 編輯配置（修改 API 密鑰）
   - 📋 查看日誌文件

### 中間日誌區域

- 實時顯示服務日誌
- 支持自動滾動
- 可以清空和刷新

### 右側信息面板

- 顯示系統信息和配置路徑

## 💡 使用示例

### 啟動服務

1. 打開服務管理器
2. 點擊「▶ 啟動服務」
3. 等待幾秒鐘，查看日誌確認啟動成功
4. 狀態顯示「● 運行中」表示啟動成功

### 訪問網頁

1. 確保服務正在運行
2. 點擊「🌐 首頁」或其他快速訪問按鈕
3. 瀏覽器會自動打開對應頁面

### 配置 API 密鑰

1. 點擊「⚙️ 編輯配置」
2. 在編輯器中修改 API 密鑰
3. 點擊「保存」
4. 點擊「↻ 重啟服務」使配置生效

## ⚠️ 常見問題

### Q: 提示「No module named 'tkinter'」

**A:** 需要安裝 Tkinter：
```bash
sudo apt-get install python3-tk
```

### Q: 服務啟動失敗

**A:** 
1. 查看日誌區域的錯誤信息
2. 確認端口 5000 未被占用
3. 檢查 `app.py` 文件是否存在

### Q: 無法打開網頁

**A:**
1. 確認服務狀態為「● 運行中」
2. 點擊「🔍 健康檢查」確認服務正常
3. 檢查防火牆設置
