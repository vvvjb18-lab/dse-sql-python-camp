# SQL & Python 训练营 - 新设计方案

## 🎯 设计理念

### 视觉风格
- **现代极简主义**：采用深色主题配合霓虹蓝绿色调
- **专业科技感**：使用渐变、毛玻璃效果和微妙的动画
- **高对比度**：确保文本清晰可读，符合无障碍设计标准

### 颜色方案
```css
/* 主色调 */
--primary-dark: #0f172a (深蓝黑背景)
--primary-blue: #00f5ff (霓虹蓝)

--accent-green: #10b981 (翠绿)
```

- **正文**：Inter Regular 16px-18px
- **代码**：JetBrains Mono 14px-16px

## 🏗️ 网站架构

### 页面结构
1. **首页 (index.html)** - 综合性学习平台入口
2. **SQL训练营 (sql-trainer.html)** - 专门的SQL学习模块
3. **Python学习中心 (python-center.html)** - Python编程教学
4. **实战项目 (projects.html)** - 综合项目练习
5. **个人档案 (profile.html)** - 学习进度和成就

### 核心功能模块

#### 1. SQL训练营模块
- **基础查询练习**：SELECT, WHERE, ORDER BY
- **高级查询**：JOIN, 子查询, 聚合函数
- **数据库设计**：表结构设计, 规范化
- **性能优化**：索引, 查询优化技巧

#### 2. Python学习中心模块  
- **基础语法**：变量, 数据类型, 控制结构
- **数据结构**：列表, 字典, 集合, 元组
- **算法实现**：排序, 搜索, 递归
- **文件处理**：文本文件, CSV处理
- **面向对象**：类, 继承, 多态

#### 3. 项目实战模块
- **Web应用开发**：Flask/Django基础项目
- **数据分析**：使用Pandas和Matplotlib
- **数据库集成**：SQLite, MySQL应用
- **自动化脚本**：实用工具开发

## 🛠️ 技术实现

### 前端技术栈
- **HTML5 + CSS3**：语义化结构，现代CSS特性
- **Tailwind CSS**：实用优先的CSS框架
- **JavaScript ES6+**：现代JavaScript语法
- **动画库**：Anime.js用于流畅动画效果
- **代码编辑器**：CodeMirror提供语法高亮
